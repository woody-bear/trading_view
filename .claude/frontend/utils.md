# Frontend Utils

`frontend/src/utils/` 는 컴포넌트·페이지·스토어 어디에서도 재사용되는 **순수 함수**를 모은다. 상태·네트워크·DOM 접근 로직은 여기에 두지 않는다.

## 파일 인벤토리

| 모듈 | 용도 |
|------|------|
| `format.ts` | 가격·금액·퍼센트·시간 포맷(KR/US 통화 분기) |
| `indicatorLabels.ts` | 기술지표(RSI/BB/EMA 등) 한글 라벨 매핑 |
| `buyReason.ts` | BUY 신호 사유를 사용자 표시용 문장으로 변환 |

## 사용 규칙

- 가격 표시는 반드시 `format.ts`의 함수를 거친다(직접 `.toLocaleString()` 금지 — 통화 분기 놓침).
- 지표 라벨은 `indicatorLabels.ts`를 단일 소스로 쓴다. 컴포넌트 안에서 직접 문자열 작성 금지.
- BUY 사유 문구 변경은 사용자가 보는 텍스트이므로 디자인·UX 검토 필요.

## 확장 규칙

- 여기에 들어가려면 **순수 함수**여야 한다(React 훅·네트워크 호출 금지).
- 2곳 이상에서 재사용될 것으로 예상될 때만 이 폴더에 둔다(단일 사용처면 해당 컴포넌트 인근에 유지).
- 새 파일 추가 시 이 문서의 인벤토리 테이블을 갱신.
