# 한국투자증권 API 실시간 가격 반영 계획

**작성일**: 2026-03-18 | **상태**: 계획

## 1. 개요

한국 주식의 실시간 체결가를 한국투자증권 Open API WebSocket으로 수신하여, 현재 pykrx(수 분 지연) 대비 **초 단위 실시간 가격**을 대시보드에 반영한다. 차트는 기존 TradingView lightweight-charts를 유지하며, 실시간 가격만 한투 API로 교체한다.

## 2. 한국투자증권 Open API 요약

| 항목 | 내용 |
|------|------|
| **비용** | 무료 (API 사용료 없음, 매매 시에만 수수료) |
| **인증** | APP_KEY + APP_SECRET → access_token 발급 |
| **실시간** | WebSocket (별도 approval_key 필요) |
| **종목 제한** | 계정당 **41종목** 동시 구독 (WebSocket) |
| **REST 제한** | 초당 ~20건 |
| **모의투자** | 무료 지원 (개발/테스트용) |
| **추천 라이브러리** | `python-kis` (Soju06) — 자동 재연결, 타입 힌트 완벽 |

## 4. 아키텍처

```
현재 (pykrx 폴링):
  APScheduler 5초 → pykrx.get_market_ohlcv() → DB 업데이트 → WebSocket broadcast
  단점: 수 분 지연, 체결가 아닌 집계 데이터

변경 후 (한투 WebSocket):
  한투 WebSocket 연결 → 실시간 체결가 수신 → DB 업데이트 → 브라우저 WebSocket broadcast
  장점: 초 단위 실시간, 체결가 직접 수신
```

### 데이터 흐름

```
[한투 WebSocket] ──체결가──→ [KISPriceFeed] ──→ [DB current_signal 업데이트]
                                            ──→ [브라우저 WebSocket broadcast]
                                            ──→ [차트 마지막 캔들 update()]

[pykrx] ── fallback (한투 API 장애 시)
[ccxt]  ── 암호화폐 (기존 유지)
[yfinance] ── 미국 (기존 유지)
```

### Task 4: WebSocket 매니저 (1시간)

**파일**: `backend/services/kis_websocket.py` (신규)

```python
class KISWebSocketManager:
    """한투 실시간 체결가 WebSocket 관리."""

    MAX_SUBSCRIPTIONS = 40  # 41개 중 1개 여유

    def __init__(self):
        self._prices: dict[str, dict] = {}  # {symbol: {price, change_pct, ...}}
        self._connected = False

    async def connect(self):
        """WebSocket 연결 + 워치리스트 종목 구독."""

    def on_price(self, symbol: str, data: dict):
        """체결가 수신 콜백 → 메모리 캐시 업데이트."""
        self._prices[symbol] = {
            "price": data["current_price"],
            "open": data["open_price"],
            "high": data["high_price"],
            "low": data["low_price"],
            "volume": data["volume"],
            "change_pct": data["change_rate"],
        }

    def get_cached_price(self, symbol: str) -> dict | None:
        return self._prices.get(symbol)
```

- 워치리스트 변경 시 구독 종목 동적 업데이트
- 41종목 제한 관리 (워치리스트 우선, 나머지 REST fallback)
- 자동 재연결 (python-kis 내장)

| 시나리오 | 대응 |
|----------|------|
| 워치리스트 ≤ 40개 | 전부 WebSocket 구독 |
| 워치리스트 > 40개 | 상위 40개 WebSocket + 나머지 REST 폴링 |
| 전체 시장 스캔 (149개) | 스캔은 yfinance 배치 (기존), 실시간은 워치리스트만 |

## 9. Fallback 체인

```
한국 주식 실시간 가격:
  1순위: 한투 WebSocket (실시간, 초 단위)
  2순위: 한투 REST API (1초 지연, rate limit 주의)
  3순위: pykrx (수 분 지연, 무료)

미국 주식: yfinance fast_info (기존 유지)
암호화폐: ccxt Binance ticker (기존 유지)
```
