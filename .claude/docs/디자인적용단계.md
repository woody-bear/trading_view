## 잔여 작업 상세 (모두 완료됨)

### 12단계 보정 — PC 종목 상세 브레이크포인트 불일치 ⚠️

**현황**: `SignalDetail.tsx:535`에서 2컬럼 레이아웃이 `md:grid` (768px)로 적용 중.  
스펙 4단계에서 `md` → `xl`(1280px)로 전환했으나 이 파일만 누락됨.

```tsx
// 현재 (잘못됨)
<div className="md:grid md:gap-3" ...>
  // 사이드레일
  <div className="flex flex-col mt-3 md:mt-0" ...>

// 수정 대상
<div className="xl:grid xl:gap-3" ...>
  <div className="flex flex-col mt-3 xl:mt-0" ...>
```

**파일**: `frontend/src/pages/SignalDetail.tsx`  
**범위**: `md:grid`, `md:gap-3`, `md:mt-0` → `xl:` 교체 (3곳)  
**난이도**: S (단순 문자열 교체)

---

### 13단계 보정 — 조회조건 Mermaid vs SVG 다이아몬드 ⚠️

**현황**: `ScanConditions.tsx`에서 `BUY_PIPELINE_MERMAID`, `SELL_FLOWCHART_MERMAID`로 Mermaid 라이브러리 기반 렌더링 중.  
스펙 13단계는 "SVG 다이아몬드 플로우차트"로 교체를 명시했으나, 027 브랜치에서 Mermaid로 먼저 구현됨.

**선택지**:
- A) **현 Mermaid 유지** — 기능 동일, 시각 디자인만 차이. 작업 불필요.
- B) **SVG 다이아몬드로 교체** — 디자인 파일(`pc-pipeline.jsx`)의 노드 구조를 직접 SVG 컴포넌트로 포팅. 상당한 작업량.

**권고**: 사용자 확인 필요. Mermaid도 충분히 읽기 가능하다면 A 선택으로 13단계 완료 처리.
