# Feature Specification: 공포지수 차트 개선

**Feature Branch**: `013-fear-index-chart`
**Created**: 2026-04-03
**Status**: Draft
**Input**: User description: "공포지수 차트를 보기좋게 만드는 기능"

## Overview

현재 홈 화면 상단 시장 심리 패널(Fear & Greed 게이지 + 30일 미니 추이 + VIX 미니카드)을 시각적으로 더 직관적이고 정보가 풍부하게 개선한다. 투자자가 시장 공포/탐욕 수준과 역사적 맥락을 한눈에 파악할 수 있도록 차트와 레이아웃을 전면 개편한다.

---

## User Scenarios & Testing

### User Story 1 — 공포지수 히스토리 차트 (Priority: P1)

투자자가 현재 Fear & Greed 수치가 역사적으로 어느 수준인지 확인하고 싶다. 현재는 30일 미니 차트만 보이는데, 기간을 선택해서 더 긴 흐름을 볼 수 있어야 한다.

**Why this priority**: 공포지수의 현재 수치가 의미 있으려면 과거 맥락이 필요하다. "지금 30이 공포인가?"를 판단하려면 3개월·1년 전 수준을 봐야 한다.

**Independent Test**: 기간 탭을 클릭하면 해당 기간의 Fear & Greed 추이 차트가 즉시 표시된다.

**Acceptance Scenarios**:

1. **Given** 시장 심리 패널이 로드됐을 때, **When** "3개월" 탭을 선택하면, **Then** 90일간 Fear & Greed 값 추이 차트가 공포/중립/탐욕 색상 배경 구간과 함께 표시된다
2. **Given** 차트가 표시된 상태에서, **When** 특정 날짜에 마우스 오버(또는 터치)하면, **Then** 해당 날짜의 수치와 레이블(예: "공포 28")이 툴팁으로 표시된다
3. **Given** 기간 탭이 "30일"로 기본 선택된 상태에서, **When** "1년" 탭을 클릭하면, **Then** 1초 이내에 1년치 데이터로 차트가 전환된다

---

### User Story 2 — 공포·탐욕 구간 색상 시각화 (Priority: P2)

차트 배경에 색상 구간을 적용해서 공포/탐욕 단계를 차트만 봐도 직관적으로 알 수 있게 한다.

**Why this priority**: 숫자 없이도 색상만으로 현재 수준을 즉각 인식할 수 있어 분석 속도가 빨라진다.

**Independent Test**: 차트 배경에 빨간색(0~25), 회색(25~75), 녹색(75~100) 구간 밴드가 그려진다.

**Acceptance Scenarios**:

1. **Given** Fear & Greed 차트가 표시된 상태에서, **When** 값이 25 이하 구간을 지나면, **Then** 해당 배경 영역이 붉은색으로 강조된다
2. **Given** 현재 값이 극도의 공포(20 이하)일 때, **When** 게이지를 보면, **Then** 게이지 침이 빨간 구역을 가리키고 수치 텍스트도 빨간색으로 표시된다

---

### User Story 3 — VIX 단독 확장 차트 (Priority: P3)

VIX(변동성 지수)를 별도로 확장해서 볼 수 있는 뷰를 제공한다. VIX 20(공포 임계값)과 VIX 30(패닉 임계값) 기준선을 차트에 표시한다.

**Why this priority**: VIX는 Fear & Greed와 별개로 매수 타이밍 판단에 자주 활용된다. VIX 20/30 수준 돌파 여부가 중요한 투자 신호이다.

**Independent Test**: VIX 카드 클릭 시 VIX 전용 차트가 열리고 20·30 수평 기준선이 점선으로 표시된다.

**Acceptance Scenarios**:

1. **Given** VIX 미니 카드가 표시된 상태에서, **When** 클릭하면, **Then** VIX 히스토리 차트가 펼쳐지고 20·30 수평 기준선이 점선으로 표시된다
2. **Given** VIX 차트가 표시된 상태에서, **When** VIX 값이 30을 초과한 구간이 있으면, **Then** 해당 구간의 차트 영역이 진한 붉은색으로 강조된다

---

### Edge Cases

- Fear & Greed 데이터를 불러오지 못할 때: 빈 차트 대신 "데이터 없음" 안내 표시
- 요청 기간보다 실제 데이터가 짧을 때: 가용 데이터 범위만 표시하고 기간 탭 비활성화
- 모바일 터치 시 세로 스크롤과 차트 가로 드래그가 충돌하지 않아야 함
- 5분 자동 갱신 시 사용자가 선택한 기간·스크롤 위치가 초기화되어서는 안 됨

---

## Requirements

### Functional Requirements

- **FR-001**: 사용자는 Fear & Greed Index 추이를 1개월 / 3개월 / 1년 기간으로 선택해서 볼 수 있어야 한다
- **FR-002**: Fear & Greed 차트 배경에는 공포(0~25 빨강), 중립(25~75 회색), 탐욕(75~100 초록) 구간이 항상 표시되어야 한다
- **FR-003**: 차트에서 날짜 포인트 선택(마우스 오버 또는 터치) 시 해당 날짜의 수치·레이블 툴팁이 표시되어야 한다
- **FR-004**: VIX 카드 클릭 시 VIX 히스토리 차트가 확장 표시되어야 하며, 수평 기준선(20·30)이 점선으로 표시되어야 한다
- **FR-005**: 현재 Fear & Greed 반원형 게이지는 유지하되, 크기·가독성·색상 정확도를 개선해야 한다
- **FR-006**: 차트는 5분마다 자동 갱신되어야 하며, 갱신 시 선택된 기간과 차트 상태가 유지되어야 한다
- **FR-007**: 모바일 화면에서도 차트를 손가락으로 조작해 값을 확인할 수 있어야 한다

### Key Entities

- **FearGreedHistory**: 날짜별 Fear & Greed 점수(0~100)와 레이블(Extreme Fear / Fear / Neutral / Greed / Extreme Greed)
- **VIXHistory**: 날짜별 VIX 종가 시계열, 기준선 수치(20: 일반 공포, 30: 패닉)
- **ChartPeriod**: 사용자 선택 기간 (1개월 / 3개월 / 1년), 마지막 선택값 유지

---

## Success Criteria

### Measurable Outcomes

- **SC-001**: 기간 탭 전환 후 1초 이내에 차트가 표시된다
- **SC-002**: 모바일과 PC 모두에서 툴팁과 기간 전환이 정상 동작한다
- **SC-003**: 공포/중립/탐욕 구간을 숫자 없이 색상만으로 즉각 식별할 수 있다
- **SC-004**: VIX 20/30 기준선이 차트에 명확히 표시되어 현재 수준 파악에 5초 이내 소요된다
- **SC-005**: 5분 자동 갱신 시 차트 위치·기간 설정이 유지된다

---

## Assumptions

- Fear & Greed 히스토리 데이터는 백엔드 `/api/sentiment/history` 엔드포인트를 기간 파라미터 지원으로 확장하는 것으로 가정
- VIX 히스토리는 yfinance `^VIX` 티커로 기간별 조회 가능한 것으로 가정
- 현재 SentimentPanel 컴포넌트를 개선하되 기존 API 응답 구조는 하위호환 유지
- 1년치 데이터는 약 252개 거래일 포인트로 성능상 문제 없음
