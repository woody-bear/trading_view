# Feature Specification: 분할매수/매도 포지션 가이드

**Feature Branch**: `007-position-guide`
**Created**: 2026-03-28
**Status**: Draft
**Input**: BUY/SELL 신호 기반 단계별 분할매수/매도 가이드를 종목 상세화면에 표시

## User Scenarios & Testing *(mandatory)*

### User Story 1 - BUY 신호 발생 시 분할매수 가이드 (Priority: P1)

종목 상세화면에서 BUY 신호가 발생하면 3단계 분할매수 가이드가 표시된다. 각 단계별로 현재 지표 충족 여부가 실시간 반영되어, 지금 어떤 행동을 해야 하는지 한눈에 파악할 수 있다.

**Why this priority**: BUY 신호만으로는 "얼마나, 언제" 매수해야 하는지 알 수 없음. 단계별 비중 안내가 있어야 실제 매매 판단으로 이어짐.

**Independent Test**: BUY 신호 종목 상세화면에서 3단계 가이드 카드가 표시되고, 현재 지표 기준으로 활성/대기 상태가 올바르게 표시되는지 확인

**Acceptance Scenarios**:

1. **Given** BUY 신호가 발생한 종목, **When** 상세화면 진입, **Then** 3단계 분할매수 가이드가 표시된다
   - 1단계: "1차 진입 30%" — 조건: RSI < 40 + BB 하단 터치
   - 2단계: "2차 추가 30%" — 조건: RSI < 30 (추가 과매도)
   - 3단계: "3차 추가 40%" — 조건: EMA20 > EMA50 (BULL 트렌드 확인)
2. **Given** 현재 RSI=35 (1단계 충족), **When** 가이드 표시, **Then** 1단계가 초록색 활성, 2~3단계는 회색 대기
3. **Given** 현재 RSI=28 + EMA20>EMA50, **When** 가이드 표시, **Then** 1~3단계 모두 활성 → "풀 포지션 진입 가능" 안내

---

### User Story 2 - SELL 신호 발생 시 분할매도 가이드 (Priority: P1)

SELL 신호가 발생하면 2단계 분할매도 가이드가 표시된다.

**Why this priority**: 매수만큼 중요한 매도 타이밍. SELL 후 전량 매도할지, 분할할지 안내.

**Independent Test**: SELL 신호 종목 상세화면에서 2단계 매도 가이드가 표시되는지 확인

**Acceptance Scenarios**:

1. **Given** SELL 신호가 발생한 종목, **When** 상세화면 진입, **Then** 2단계 분할매도 가이드가 표시된다
   - 1단계: "1차 매도 50%" — 조건: RSI > 65 + BB 상단
   - 2단계: "잔여 전량 매도" — 조건: RSI > 70 (추가 과매수)
2. **Given** 현재 RSI=72, **When** 가이드 표시, **Then** 1~2단계 모두 활성 → "전량 매도 검토" 강조

---

### User Story 3 - NEUTRAL 상태 관망 가이드 (Priority: P2)

BUY도 SELL도 아닌 NEUTRAL 상태에서 왜 기다려야 하는지 설명하고, 다음 신호 발생까지 어떤 지표를 주시해야 하는지 안내한다.

**Why this priority**: 신호가 없을 때도 맥락을 제공하여 불필요한 행동 방지.

**Independent Test**: NEUTRAL 종목에서 관망 안내가 표시되고, BUY/SELL 조건까지의 거리가 표시되는지 확인

**Acceptance Scenarios**:

1. **Given** NEUTRAL 상태 종목, **When** 상세화면 진입, **Then** "매수·매도 조건 미충족 — 신호 대기 중" 안내 표시
2. **Given** NEUTRAL에서 RSI=45, **When** 가이드 표시, **Then** "RSI 40 이하 진입 시 BUY 가이드 활성화" 힌트 표시

---

### Edge Cases

- 관심종목에 등록되지 않은 종목은 차트 지표 데이터에서 RSI/BB/EMA를 직접 참조하여 가이드 표시
- 미국 주식/암호화폐도 동일한 가이드 규칙 적용
- 장 마감 후에는 마지막 봉 기준 지표로 가이드 상태 유지
- 가이드 하단에 면책 문구 필수 포함: "투자 참고 정보이며, 투자 판단은 본인 책임입니다"

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: BUY 신호 종목에서 3단계 분할매수 가이드를 표시해야 한다 (1차 30% / 2차 30% / 3차 40%)
- **FR-002**: SELL 신호 종목에서 2단계 분할매도 가이드를 표시해야 한다 (1차 50% / 잔여 전량)
- **FR-003**: 각 단계별 조건 충족 여부를 활성(초록)/대기(회색)로 시각화해야 한다
- **FR-004**: 매수 단계별 조건: 1차(RSI<40 + BB하단), 2차(RSI<30), 3차(EMA20>EMA50)
- **FR-005**: 매도 단계별 조건: 1차(RSI>65 + BB상단), 2차(RSI>70)
- **FR-006**: NEUTRAL 상태에서 관망 안내와 다음 신호 조건까지의 거리를 표시해야 한다
- **FR-007**: 가이드에 현재 지표 수치(RSI, BB %B, EMA 상태)를 함께 표시해야 한다
- **FR-008**: 면책 문구를 가이드 하단에 표시해야 한다
- **FR-009**: 가이드는 차트 아래, 투자지표 위에 배치해야 한다

### Key Entities

- **포지션 가이드**: 신호 상태(BUY/SELL/NEUTRAL)에 따른 단계별 행동 안내
- **단계(Step)**: 비중(%), 조건, 활성/대기 상태
- **지표 기준값**: RSI 30/40/65/70, BB %B 하단(0.2)/상단(0.8), EMA20 vs EMA50

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: BUY/SELL 신호 종목에서 가이드가 1초 이내에 표시된다
- **SC-002**: 각 단계의 활성/대기 상태가 현재 지표에 정확히 연동된다
- **SC-003**: NEUTRAL 종목에서 관망 안내가 표시된다
- **SC-004**: 면책 문구가 모든 가이드에 포함된다
- **SC-005**: 가이드 표시가 기존 차트/지표 기능에 영향을 주지 않는다

## Assumptions

- 비중(30/30/40, 50/전량)과 지표 기준값은 고정 — 사용자 커스터마이징 없음
- 실제 매매 실행 기능 없음 — 화면상 안내만 제공
- 신호 상태는 기존 current_signal 또는 차트 마지막 마커에서 가져옴
- 가이드는 프론트엔드 전용 로직 — 백엔드 변경 최소화 (기존 지표 데이터 활용)
