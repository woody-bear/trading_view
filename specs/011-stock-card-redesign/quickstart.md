# Quickstart: 종목 카드 라벨 & 컬러 테마 검증

## 시나리오 1: 관심종목 카드 라벨 확인

1. 관심종목에 KOSPI BUY 종목 추가
2. 메인페이지 열기
3. 확인:
   - `KOSPI` 배지 표시됨
   - `STRONG BUY` 또는 `BUY` 라벨 표시됨
   - RSI < 30이면 `RSI 과매도` 라벨 표시됨
   - 지표 라벨 최대 4개 이하

## 시나리오 2: 차트 BUY 신호 카드 라벨 확인

1. 메인페이지 전체 시장 스캔 실행
2. 차트 BUY 신호 섹션 확인:
   - `NQ100` 또는 `S&P500` 배지 표시됨
   - `SQZ BUY` 종목: `스퀴즈해소` 라벨 (시안)
   - `BB BUY` 종목: `BB반전` 라벨 (녹색)
   - `MAX SQ` 있으면 빨간 라벨 표시됨

## 시나리오 3: 투자과열 카드 라벨 확인

1. 투자과열 섹션 펼치기
2. 확인:
   - `KOSPI` 또는 `KOSDAQ` 배지 표시됨
   - RSI ≥ 80: `RSI 극과매수` (빨강)
   - RSI 70~80: `RSI 과매수` (주황)
   - 거래량 ≥ 3x: `거래량 폭증` (보라)

## 시나리오 4: 컬러 테마 검증

1. 사이트 전체 페이지 순회 (메인, 스캔, 설정, 스크랩, BUY리스트)
2. 확인:
   - 배경이 기존보다 밝은 네이비 톤
   - 카드가 배경보다 명확히 밝아 구분됨
   - 텍스트 가독성 저하 없음
   - 어두운 환경에서 눈부심 없음

## 시나리오 5: 라벨 없는 종목 레이아웃

1. 지표 조건 미충족 NEUTRAL 종목 (BUY/SELL 아님)
2. 확인:
   - 시장 유형 배지만 표시됨 (고정)
   - 지표 라벨 없어도 카드 레이아웃 정상
   - 빈 라벨 행이 불필요한 공간 차지 안 함

## 시나리오 6: 모바일 레이아웃

1. Chrome DevTools 375px 에뮬레이션
2. 종목 카드 스크롤
3. 확인:
   - 종목명 잘리지 않음
   - 라벨들 카드 밖으로 넘치지 않음
   - 라벨 줄바꿈 자연스럽게 처리됨
