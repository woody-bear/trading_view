# Feature Specification: 종목 상세화면 업그레이드 (한국투자증권 API)

**Feature Branch**: `006-kis-stock-detail`
**Created**: 2026-03-26
**Status**: Draft
**Input**: 한국투자증권 API에서 불러올 수 있는 유용한 정보(EPS, PER, PBR, 52주 고/저, 호가, 시가총액 등)를 활용하여 종목 상세화면을 업그레이드

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 기본 투자지표 확인 (Priority: P1)

사용자가 종목 상세화면에 진입하면, 차트 아래에 해당 종목의 핵심 투자지표(PER, PBR, EPS, BPS, 시가총액, 업종)를 즉시 확인할 수 있다. 이를 통해 해당 종목이 저평가/고평가 상태인지 빠르게 판단한다.

**Why this priority**: BUY 신호가 뜬 종목의 기본 가치 판단에 가장 핵심적인 정보. 기술적 지표(RSI, BB 등)만으로는 투자 판단이 불완전하며, 펀더멘탈 데이터가 함께 있어야 의사결정이 가능.

**Independent Test**: 한국 주식 종목 상세화면에 진입하여 PER/PBR/EPS/BPS 카드가 표시되고, 수치가 정상 범위인지 확인

**Acceptance Scenarios**:

1. **Given** 한국 주식 종목 상세화면, **When** 페이지 로드 완료, **Then** PER, PBR, EPS, BPS, 시가총액, 업종이 표시된다
2. **Given** 투자지표가 표시된 상태, **When** PER이 업종 평균 대비 낮으면, **Then** "저평가" 등 시각적 힌트를 제공한다
3. **Given** KIS API 연동이 안 된 환경, **When** 종목 상세화면 진입, **Then** 지표 영역에 "KIS API 미연동" 안내를 표시하고 나머지 화면은 정상 동작한다

---

### User Story 2 - 52주 고가/저가 현재 위치 확인 (Priority: P1)

사용자가 종목 상세화면에서 현재가가 52주 범위에서 어디쯤 위치하는지 시각적으로 확인할 수 있다. 52주 최고가와 최저가 날짜도 함께 표시된다.

**Why this priority**: 현재가의 상대적 위치를 한눈에 파악하는 가장 직관적인 방법. BUY 신호와 결합하면 "52주 저점 근처에서 BUY 신호" 같은 고가치 정보 제공.

**Independent Test**: 종목 상세화면에서 52주 범위 바가 표시되고, 현재가 위치가 정확히 표시되는지 확인

**Acceptance Scenarios**:

1. **Given** 한국 주식 상세화면, **When** 페이지 로드 완료, **Then** 52주 최고/최저가와 현재가 위치를 시각적 바로 표시한다
2. **Given** 52주 범위 표시 상태, **When** 현재가가 52주 저점 근처(하위 20%), **Then** "52주 저점 근처" 라벨을 강조 표시한다

---

### User Story 3 - 매도/매수 호가창 확인 (Priority: P2)

사용자가 종목 상세화면에서 실시간 매도/매수 호가(상위 5단계)를 확인할 수 있다. 매도/매수 물량의 비율을 시각적으로 보여주어 매수세/매도세를 직관적으로 파악한다.

**Why this priority**: 실시간 수급 상황을 파악하여 매수 타이밍을 잡는 데 유용. 단, 장 중에만 의미가 있고 기본 지표보다 우선순위가 낮음.

**Independent Test**: 장 중에 한국 주식 상세화면에서 호가 데이터가 표시되고, 매수/매도 잔량 바가 시각화되는지 확인

**Acceptance Scenarios**:

1. **Given** 장 중 한국 주식 상세화면, **When** 페이지 로드 완료, **Then** 매도 5호가 + 매수 5호가가 가격/잔량과 함께 표시된다
2. **Given** 호가 표시 상태, **When** 매수 총잔량이 매도 총잔량보다 크면, **Then** "매수 우세" 라벨과 매수/매도 비율을 표시한다
3. **Given** 장 마감 후, **When** 종목 상세화면 진입, **Then** 마지막 호가를 표시하고 "장 마감" 안내를 보여준다

---

### User Story 4 - 투자 위험 경고 표시 (Priority: P2)

종목이 매매정지, 과열종목 지정, 투자경고/위험/주의 상태일 때 상세화면 상단에 명확한 경고를 표시한다.

**Why this priority**: 투자 손실 방지를 위한 안전장치. BUY 신호가 떠도 매매정지나 투자경고 종목은 위험할 수 있으므로 반드시 알려야 함.

**Independent Test**: 매매정지 또는 투자경고 종목 상세화면에서 경고 배너가 표시되는지 확인

**Acceptance Scenarios**:

1. **Given** 매매정지 종목, **When** 상세화면 진입, **Then** 빨간색 "매매정지" 경고 배너를 차트 상단에 표시한다
2. **Given** 과열종목 지정 종목, **When** 상세화면 진입, **Then** 주황색 "과열종목" 경고를 표시한다
3. **Given** 정상 종목, **When** 상세화면 진입, **Then** 경고 배너 없이 정상 표시된다

---

### User Story 5 - 가격제한 정보 표시 (Priority: P3)

상한가/하한가/기준가 정보를 종목 상세화면에 표시하여 당일 가격 변동 범위를 파악한다.

**Why this priority**: 유용하지만 필수는 아닌 보조 정보. 상한가/하한가 근처인 종목의 매매 판단에 도움.

**Independent Test**: 종목 상세화면에서 상한가/하한가/기준가가 올바르게 표시되는지 확인

**Acceptance Scenarios**:

1. **Given** 한국 주식 상세화면, **When** 페이지 로드 완료, **Then** 기준가, 상한가, 하한가를 표시한다
2. **Given** 현재가가 상한가의 90% 이상, **When** 상세화면 표시, **Then** "상한가 근접" 강조 표시

---

### Edge Cases

- KIS API 키가 설정되지 않은 환경에서는 KIS 데이터 영역을 숨기거나 "미연동" 안내를 표시
- 미국 주식/암호화폐 종목에서는 KIS 전용 지표(호가, 상한가 등)를 숨김
- ETF 종목은 EPS/PER 등이 일반 주식과 다를 수 있으므로 표시 여부를 적절히 처리
- 장 시작 전(08:00~09:00) 동시호가 시간에는 호가 데이터가 다를 수 있음
- 신규 상장 종목은 52주 데이터가 부족할 수 있음 — 가용 범위만 표시
- API 호출 실패 시 차트 및 기존 지표는 정상 동작하고, KIS 추가 데이터만 에러 처리

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: 한국 주식 상세화면에서 PER, PBR, EPS, BPS를 표시해야 한다
- **FR-002**: 한국 주식 상세화면에서 시가총액을 표시해야 한다
- **FR-003**: 한국 주식 상세화면에서 업종명을 표시해야 한다
- **FR-004**: 한국 주식 상세화면에서 52주 최고가/최저가 및 해당 날짜를 표시해야 한다
- **FR-005**: 52주 범위에서 현재가의 상대적 위치를 시각적 바(progress bar)로 표시해야 한다
- **FR-006**: 한국 주식 상세화면에서 매도/매수 각 5단계 호가와 잔량을 표시해야 한다
- **FR-007**: 매수/매도 총잔량 비율을 시각적으로 표시해야 한다
- **FR-008**: 매매정지, 과열종목, 투자경고/위험/주의 상태를 경고 배너로 표시해야 한다
- **FR-009**: 기준가, 상한가, 하한가를 표시해야 한다
- **FR-010**: KIS API 미연동 시 해당 섹션을 graceful하게 처리해야 한다 (기존 기능에 영향 없음)
- **FR-011**: 미국 주식/암호화폐 종목에서는 KIS 전용 지표를 숨겨야 한다
- **FR-012**: 투자지표 데이터를 캐싱하여 동일 종목 재조회 시 즉시 표시해야 한다

### Key Entities

- **종목 투자지표**: PER, PBR, EPS, BPS, 시가총액, 업종명, 52주 고/저가 — 종목의 기본 가치 판단 정보
- **호가 데이터**: 매도 5호가, 매수 5호가, 각 호가별 잔량, 총잔량 — 실시간 수급 정보
- **종목 상태**: 매매정지 여부, 과열종목 지정, 투자경고/위험/주의 등급 — 리스크 정보
- **가격제한**: 기준가, 상한가, 하한가 — 당일 가격 변동 범위

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 한국 주식 상세화면에서 투자지표(PER/PBR/EPS/BPS/시가총액/업종)가 1초 이내에 표시된다
- **SC-002**: 52주 고/저가 범위와 현재가 위치가 시각적으로 즉시 파악 가능하다
- **SC-003**: 장 중 호가 데이터가 실시간으로 갱신되어 표시된다
- **SC-004**: 매매정지/투자경고 종목 진입 시 경고가 0.5초 이내에 눈에 띄게 표시된다
- **SC-005**: KIS API 미연동 환경에서도 기존 차트/지표 기능이 100% 정상 동작한다
- **SC-006**: 미국 주식/암호화폐 상세화면에서 KIS 전용 영역이 표시되지 않아 혼란이 없다

## Assumptions

- 한국투자증권 API(pykis)의 `stock.quote()` 응답에 EPS, BPS, PER, PBR, 52주 고/저, 시가총액, 업종명, 매매정지/과열/경고 상태가 포함됨
- `stock.orderbook()` 응답에 매도/매수 각 5단계 호가+잔량이 포함됨
- KIS API 호출 빈도 제한은 초당 20회 이내로 충분
- 투자지표(PER 등)는 장 중에도 크게 변동하지 않으므로 5분 캐싱이 적절
- 호가 데이터는 실시간성이 중요하므로 캐싱 없이 매 요청 시 조회하거나 짧은 TTL(5초) 적용
